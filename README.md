# AI-Backend-Dev

--------------------
Prosjektbeskrivelse
--------------------
Dette prosjektet er en AI-basert backend-løsning utviklet i Python. Systemet lar brukere sende spørringer til en AI-modell via et API. Spørringene og svarene fra AI blir lagret i en SQL-database for loggføring og sporbarhet.

Prosjektet kombinerer:
  - Backend-utvikling
  - Databasehåndtering
  - AI-integrasjon
  - Logging og driftsperspektiv

--------------------
Målet
--------------------
- Sette opp en AI-modell
- Sende spørringer til AI via API
- Motta svar fra AI
- Lagre både spørring og svar i SQL-database
- Loggføre all bruk av AI
- Strukturere løsningen som en profesjonell backend
